Otomasi Telegram dan Alert

n8n menjadi jembatan antara MQTT dan Telegram, sehingga data sensor bisa dikirim sebagai notifikasi.

Data MQTT

Payload JSON dari ESP memakai data HC-SR04 pada field distance_cm. Jarak besar berarti air rendah, jarak kecil berarti air cukup atau penuh.

Alert

Alert utama mengikuti logika HC-SR04: distance_cm >= 15 berarti air rendah, sedangkan distance_cm <= 8 berarti air cukup atau penuh.

Command

Pesan Telegram seperti ON, OFF, AUTO, dan RESET_WIFI dipetakan ke topic MQTT control.

Function Node: Format Monitoring

const data = JSON.parse($json.message || "{}");

const temperature = data.temperature ?? "-";
const humidity = data.humidity ?? "-";
const distanceCm = data.distance_cm ?? "-";
const levelAirPersen = data.water_level_percent ?? "-";
const pumpStatus = data.pump_status ?? "-";
const pumpMode = data.pump_mode ?? "-";

let alert = "";

if (Number(distanceCm) >= 15) {
  alert += "\nPERINGATAN: Jarak air tinggi, air rendah.";
}

if (Number(distanceCm) <= 8) {
  alert += "\nINFO: Air cukup / penuh.";
}

if (Number(temperature) >= 34) {
  alert += "\nPERINGATAN: Suhu terlalu tinggi.";
}

return [
  {
    json: {
      text:
        "Smart Hydroponic Monitoring\n" +
        "Suhu: " + temperature + " C\n" +
        "Kelembaban: " + humidity + " %\n" +
        "Jarak HC-SR04: " + distanceCm + " cm\n" +
        "Level air: " + levelAirPersen + " %\n" +
        "Pompa: " + pumpStatus + "\n" +
        "Mode: " + pumpMode +
        alert
    }
  }
];

Function Node: Telegram Command ke MQTT

const text = ($json.message?.text || "").trim().toUpperCase();

let command = "";

if (text === "/ON" || text === "ON" || text === "NYALA") {
  command = "ON";
} else if (text === "/OFF" || text === "OFF" || text === "MATI") {
  command = "OFF";
} else if (text === "/AUTO" || text === "AUTO") {
  command = "AUTO";
} else if (text === "/RESET_WIFI" || text === "RESET_WIFI") {
  command = "RESET_WIFI";
} else {
  command = "UNKNOWN";
}

return [
  {
    json: {
      topic: "hanif/smarthydro/control",
      message: command
    }
  }
];

Penjelasan Detail Coding

JSON.parse($json.message || "{}")

n8n menerima payload MQTT terbaru dari ESP sebagai teks JSON. JSON.parse() mengubah teks tersebut menjadi object JavaScript. Jika pesan kosong, {} dipakai sebagai cadangan agar node tidak error.

data.temperature ?? "-"

Baris ini mengambil suhu dari field temperature yang dikirim oleh kirim_mqtt(). Operator ?? berarti jika nilai tidak ada atau null, tampilkan tanda minus.

data.distance_cm

Field ini berasal dari sensor HC-SR04/ultrasonic pada ESP8266. Inilah data utama untuk menentukan kondisi air: makin besar jaraknya, makin rendah permukaan air.

Variabel alert

Variabel ini awalnya kosong. Jika jarak ultrasonic menunjukkan air rendah atau suhu terlalu tinggi, teks peringatan ditambahkan ke pesan monitoring.

Number(distanceCm) >= 15

Ini mengikuti batas pada ESP: jika jarak HC-SR04 lebih besar atau sama dengan 15 cm, permukaan air jauh dari sensor, berarti air rendah.

Number(distanceCm) <= 8

Ini juga mengikuti batas pada ESP: jika jarak kurang dari atau sama dengan 8 cm, permukaan air dekat dengan sensor, berarti air cukup atau penuh.

Number(temperature) >= 34

Suhu diubah menjadi angka dan dibandingkan dengan batas 34 C. Jika suhu melewati batas, pesan Telegram diberi peringatan suhu tinggi.

return [{ json: { text: ... } }]

n8n memakai format output berbentuk array item. Field text nanti bisa langsung dipakai oleh node Telegram untuk isi pesan.

$json.message?.text

Bagian ini mengambil teks dari pesan Telegram. Tanda ?. mencegah error jika message tidak ada.

trim().toUpperCase()

trim() menghapus spasi, sedangkan toUpperCase() membuat command menjadi huruf besar. Dengan begitu command on, ON, dan On tetap dikenali.

Mapping Command

Blok if dan else if mengubah pesan Telegram menjadi command standar: ON, OFF, AUTO, RESET_WIFI, atau UNKNOWN.

topic: "hanif/smarthydro/control"

Output ini dikirim ke node MQTT Publish. Topic harus sama dengan topic yang disubscribe ESP agar command sampai ke perangkat.
