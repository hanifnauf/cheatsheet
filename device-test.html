1. Test HC-SR04 / Ultrasonic

Catatan Wiring

Pin ECHO HC-SR04 harus memakai pembagi tegangan karena output ECHO 5V, sedangkan ESP8266 hanya aman menerima 3.3V.

import time
import machine
from machine import Pin

TRIG_PIN = Pin(16, Pin.OUT)  # D0 / GPIO16
ECHO_PIN = Pin(4, Pin.IN)    # D2 / GPIO4

def baca_jarak_once():
    TRIG_PIN.value(0)
    time.sleep_us(2)

    TRIG_PIN.value(1)
    time.sleep_us(10)
    TRIG_PIN.value(0)

    durasi = machine.time_pulse_us(ECHO_PIN, 1, 30000)

    if durasi < 0:
        return None

    jarak_cm = durasi / 58
    return jarak_cm

while True:
    jarak = baca_jarak_once()

    if jarak is None:
        print("Ultrasonic gagal membaca jarak")
    else:
        print("Jarak HC-SR04:", round(jarak, 1), "cm")

    time.sleep(1)

Penjelasan HC-SR04

TRIG_PIN dan ECHO_PIN

TRIG_PIN mengirim sinyal pemicu 10 microsecond. ECHO_PIN menerima pantulan suara dari sensor.

machine.time_pulse_us(ECHO_PIN, 1, 30000)

Baris ini menghitung lama sinyal ECHO bernilai HIGH. Timeout 30000 microsecond mencegah program menunggu terlalu lama jika sensor tidak merespons.

durasi / 58

Rumus ini mengubah waktu pantulan ultrasonic menjadi jarak dalam cm. Jika jarak makin besar, berarti permukaan air makin jauh dari sensor.

2. Test DHT11

import time
import dht
from machine import Pin

DHT_PIN = Pin(14)  # D5 / GPIO14
dht_sensor = dht.DHT11(DHT_PIN)

while True:
    try:
        dht_sensor.measure()
        suhu = dht_sensor.temperature()
        kelembaban = dht_sensor.humidity()

        print("Suhu:", suhu, "C")
        print("Kelembaban:", kelembaban, "%")
        print("--------------------")

    except Exception as e:
        print("Gagal membaca DHT11:", e)

    time.sleep(2)

Penjelasan DHT11

dht.DHT11(DHT_PIN)

Membuat object sensor DHT11 pada pin D5/GPIO14. Object ini dipakai untuk membaca suhu dan kelembaban.

dht_sensor.measure()

Meminta sensor melakukan pembacaan baru. Setelah baris ini, nilai suhu dan kelembaban bisa diambil.

time.sleep(2)

DHT11 tidak perlu dibaca terlalu cepat. Jeda 2 detik membuat pembacaan lebih stabil.

3. Test Relay Pompa

Keamanan

Jika relay terhubung ke pompa atau tegangan tinggi, pastikan wiring COM/NO benar dan jangan menyentuh kabel saat alat menyala.

import time
from machine import Pin

RELAY_PIN = Pin(5, Pin.OUT)  # D1 / GPIO5

RELAY_ON = 0
RELAY_OFF = 1

while True:
    print("Relay ON")
    RELAY_PIN.value(RELAY_ON)
    time.sleep(3)

    print("Relay OFF")
    RELAY_PIN.value(RELAY_OFF)
    time.sleep(3)

Penjelasan Relay

RELAY_ON = 0

Relay module pada project ini aktif LOW. Nilai 0 menyalakan relay, sedangkan nilai 1 mematikan relay.

time.sleep(3)

Relay dinyalakan 3 detik dan dimatikan 3 detik agar perubahan mudah diamati dari suara klik relay atau LED indikator relay.

4. Test LED Indikator

import time
from machine import Pin

LED_MQTT = Pin(12, Pin.OUT)       # D6 / GPIO12
LED_POMPA_OFF = Pin(13, Pin.OUT)  # D7 / GPIO13
LED_POMPA_ON = Pin(15, Pin.OUT)   # D8 / GPIO15

while True:
    print("Test LED MQTT")
    LED_MQTT.value(1)
    LED_POMPA_OFF.value(0)
    LED_POMPA_ON.value(0)
    time.sleep(1)

    print("Test LED Pompa OFF")
    LED_MQTT.value(0)
    LED_POMPA_OFF.value(1)
    LED_POMPA_ON.value(0)
    time.sleep(1)

    print("Test LED Pompa ON")
    LED_MQTT.value(0)
    LED_POMPA_OFF.value(0)
    LED_POMPA_ON.value(1)
    time.sleep(1)
