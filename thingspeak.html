Kirim Monitoring ke Dashboard

ThingSpeak menerima nilai melalui URL update, lalu menampilkannya sebagai grafik per field.

Field 1-3

field1 untuk suhu, field2 untuk kelembaban, dan field3 untuk level air.

Field 4-5

field4 menyimpan status pompa sebagai angka. field5 menyimpan mode pompa sebagai angka.

Interval

Pada main.py terbaru, pengiriman ThingSpeak mengikuti INTERVAL_SEND = 20 detik.

import socket

WRITE_API_KEY = "SHKXBHKDZLDLFT2X"
THINGSPEAK_HOST = "api.thingspeak.com"

def kirim_thingspeak():
    pompa_value = 1 if status_pompa == "ON" else 0
    mode_value = 1 if mode_pompa == "AUTO" else 0

    path = f"/update?api_key={WRITE_API_KEY}&field1={suhu}&field2={kelembaban}&field3={level_air_persen}&field4={pompa_value}&field5={mode_value}"

    s = None

    try:
        print("\nMengirim ke ThingSpeak...")
        addr = socket.getaddrinfo(THINGSPEAK_HOST, 80)[0][-1]
        s = socket.socket()
        s.settimeout(5)
        s.connect(addr)

        request = f"GET {path} HTTP/1.1\r\nHost: {THINGSPEAK_HOST}\r\nConnection: close\r\n\r\n"
        s.send(request.encode())
        s.recv(128)

        print("ThingSpeak terkirim")

    except Exception as e:
        print("ThingSpeak gagal:", e)

    finally:
        if s:
            try:
                s.close()
            except Exception:
                pass

Penjelasan Detail Coding

import socket

Socket dipakai untuk membuat koneksi HTTP manual dari ESP8266 ke server ThingSpeak. Cara ini ringan dan cocok untuk MicroPython.

WRITE_API_KEY

API key ini adalah kunci channel ThingSpeak. ESP harus mengirim key tersebut agar data masuk ke channel yang benar.

pompa_value dan mode_value

Status pompa dan mode pompa diubah menjadi angka supaya mudah dibuat grafik. Pompa ON menjadi 1, OFF menjadi 0. Mode AUTO menjadi 1, MANUAL menjadi 0.

path = f"/update?..."

Versi terbaru memakai f-string. Nilai sensor langsung dimasukkan ke URL ThingSpeak: suhu, kelembaban, level air, status pompa, dan mode pompa.

socket.getaddrinfo(THINGSPEAK_HOST, 80)

Fungsi ini mencari alamat IP dari domain api.thingspeak.com pada port 80 agar ESP tahu server tujuan.

s.settimeout(5)

Timeout 5 detik mencegah ESP terlalu lama menunggu jika internet atau server sedang bermasalah.

request = f"GET {path} ..."

Request HTTP GET dibuat lengkap dengan path, host, dan connection close. Setelah request dikirim, ThingSpeak membaca parameter field dari URL.

s.send() dan s.recv(128)

s.send() mengirim request ke ThingSpeak. s.recv(128) membaca sebagian respons server sebagai tanda bahwa server sudah membalas.

finally dan s.close()

Bagian finally memastikan socket ditutup setelah selesai, baik pengiriman berhasil maupun gagal. Ini penting agar memori ESP8266 tidak cepat penuh.
